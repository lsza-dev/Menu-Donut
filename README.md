# Radial Menu

Radial Menu is a wrapper for making easelly radial menu.

![Capture](./Capture.PNG)

```js
const config = [
    {
        label:"😀 button 1",
        function:() => {}
    }
    {,
        label:"😁 button 2",
        function:() => {}
    },
    {
        label:"😂 button 3",
        function:() => {}
    },
    {
        label:"🤣 button 4",
        function:() => {}
    },
    {
        label:"😃 button 5",
        function:() => {}
    },
    {
        label:"😄 button 6",
        function:() => {}
    },
    {
        label:"😅 button 7",
        function:() => {}
    },
    {
        label:"😆 button 8",
        function:() => {}
    }
];

const radial = new RadialMenu(config);
```
